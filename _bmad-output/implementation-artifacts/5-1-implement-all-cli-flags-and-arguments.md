# Story 5.1: Implement All CLI Flags and Arguments
## Story Statement
As a developer, I want all CLI flags from Python Mistral Vibe to be available, so that scripts using roro work without modification.
## Acceptance Criteria
**Given** `roro -p "Hello" --max-turns 5 --max-price 0.10 --enabled-tools "bash:read" --output text --agent default --workdir /tmp --trust`, **When** executed, **Then** all flags parsed correctly and conversation respects limits.
**Given** `roro --version`, **When** executed, **Then** version displayed and exit code 0.
**Given** `roro --help`, **When** executed, **Then** full help text with all flags, descriptions, examples.
